# Memory Profiling Skill

Identify memory leaks, inefficiencies, and optimization opportunities in running applications.

## Description

Specialized memory profiling analyzing allocation patterns, heap usage, and garbage collection to pinpoint memory issues.

## What's Included

- **Examples**: Memory leak detection, allocation analysis
- **Reference**: Profiling tools, optimization techniques
- **Templates**: Profiling report structures

## Use When

- Memory usage growing over time
- High memory consumption detected
- Performance degradation suspected
