# Pre-Publish Checklist

Universal quality checks before publishing any written content.

## Content Quality

### Clarity
- [ ] Main point clear within first paragraph
- [ ] Each section has one clear purpose
- [ ] No jargon without explanation
- [ ] Passes "so what?" test for every section

### Structure
- [ ] Logical flow from section to section
- [ ] Transitions guide reader smoothly
- [ ] Opening hooks attention
- [ ] Ending provides closure or call to action

### Accuracy
- [ ] Facts verified from reliable sources
- [ ] Statistics have citations
- [ ] Quotes are accurate and attributed
- [ ] Names and titles spelled correctly

---

## Writing Quality

### Voice & Tone
- [ ] Consistent throughout
- [ ] Appropriate for audience
- [ ] Authentic to brand/author

### Sentences
- [ ] Varied length (mix short and long)
- [ ] Varied structure (not all same pattern)
- [ ] Active voice preferred
- [ ] No run-on sentences

### Words
- [ ] Removed unnecessary "very," "really," "just"
- [ ] Replaced vague words with specific ones
- [ ] Cut redundancies ("past history" → "history")
- [ ] Eliminated clichés where possible

---

## Technical Quality

### Grammar & Punctuation
- [ ] Subject-verb agreement
- [ ] Consistent tense
- [ ] Proper comma usage
- [ ] Quotation marks correct

### Formatting
- [ ] Consistent heading hierarchy
- [ ] Paragraphs not walls of text
- [ ] Lists formatted consistently
- [ ] White space for readability

### Links & Media
- [ ] All links working
- [ ] Images have alt text
- [ ] Media loads properly
- [ ] External links open in new tab

---

## Genre-Specific

### Blog Posts
- [ ] SEO title optimized
- [ ] Meta description written (150-160 chars)
- [ ] Primary keyword in first 100 words
- [ ] Internal links to related content
- [ ] Featured image selected

### Research/Academic
- [ ] Abstract complete and accurate
- [ ] All citations properly formatted
- [ ] References list complete
- [ ] Methodology reproducible
- [ ] Limitations acknowledged

### Fiction
- [ ] POV consistent
- [ ] Character voices distinct
- [ ] Scene goals clear
- [ ] Pacing appropriate
- [ ] Hooks at chapter ends

### Marketing Copy
- [ ] Single clear CTA
- [ ] Benefits over features
- [ ] Social proof included
- [ ] Objections addressed
- [ ] Mobile formatting checked

---

## Final Checks

### Read Aloud
- [ ] Read entire piece aloud
- [ ] Marked awkward passages
- [ ] Fixed rhythm issues

### Fresh Eyes
- [ ] Stepped away before final review
- [ ] Had someone else review (if possible)
- [ ] Checked on different device

### Publication Ready
- [ ] Title finalized
- [ ] Author/byline correct
- [ ] Publication date set
- [ ] Category/tags selected
- [ ] Backup saved

---

*Quality content reflects thorough preparation*
