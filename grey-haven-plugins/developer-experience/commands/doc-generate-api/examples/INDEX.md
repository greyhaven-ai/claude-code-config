# API Documentation Generation Examples

Complete examples demonstrating automatic API documentation generation from TypeScript and Python codebases.

## Available Examples

### [typescript-route-extraction.md](typescript-route-extraction.md)
Extract API documentation from TypeScript codebases.
- Hono route extraction (Cloudflare Workers)
- TanStack Start server functions
- NestJS decorator-based routes
- Type extraction from interfaces
- JSDoc comment parsing
- Request/response schema generation
- Authentication middleware detection

### [python-fastapi-extraction.md](python-fastapi-extraction.md)
Extract API documentation from Python FastAPI applications.
- FastAPI route decorator extraction
- Pydantic model schema generation
- Docstring parsing (Google, NumPy styles)
- Response model extraction
- Dependency injection detection
- Authentication scheme extraction
- Error response documentation

### [openapi-output-example.md](openapi-output-example.md)
Complete OpenAPI 3.1 specification output example.
- Full OpenAPI structure (paths, components, security)
- Component schemas with examples
- Request/response schemas
- Error response patterns
- Security schemes (Bearer, OAuth2)
- Multi-language code examples
- Swagger UI integration

### [coverage-report-example.md](coverage-report-example.md)
Documentation coverage analysis report example.
- Coverage metrics by endpoint
- Missing documentation identification
- Code example coverage
- Schema completeness analysis
- Actionable recommendations
- Threshold configuration
