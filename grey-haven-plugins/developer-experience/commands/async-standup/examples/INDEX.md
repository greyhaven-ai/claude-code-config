# Async Standup Examples

Complete examples demonstrating async standup generation across different formats and timeframes.

## Available Examples

### [output-formats.md](output-formats.md)
Complete examples of all 4 output formats with real data.
- **Slack Format** - Full example with emojis, sections, and threading
- **Markdown Format** - Comprehensive format with detailed metrics and impact analysis
- **Linear Comment Format** - Concise format optimized for Linear projects
- **Email Format** - Plain-text format suitable for email distribution

### [daily-standup-example.md](daily-standup-example.md)
Complete daily standup example showing all sections.
- Yesterday's accomplishments from Git commits
- Today's planned work from Linear backlog
- Active blockers with severity classification
- Productivity metrics (commits, PRs, issues, coverage)
- Smart work planning and capacity allocation

### [weekly-summary-example.md](weekly-summary-example.md)
Weekly summary format for sprint reviews.
- Week-long accomplishment summary
- Grouped by project or category
- Weekly velocity and metrics trends
- Sprint goal progress tracking
- Team collaboration highlights
